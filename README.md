# Soldeer Minimal

**Minimal Forge, Soldeer, and Bun template.**

Soldeer Minimal uses Soldeer package manager instead of git modules and can run TypeScript out of box.

## Usage

**Install**

```shell
soldeer install & bun install
```

**Build**

```shell
forge build
```

**Test**

```shell
forge test
```
